Hexo推送到github-page后没有theme

发表于 2019-04-27

终于把环境搭建好了，但是当把环境部署到github page上后，发现页面没有theme，但是我检查了确定是配置了theme的。

github链接上的预览图没有样式

没有样式不对。

检查了_config.yml文件，theme: next，这是我新下的一个theme主题，工程没问题。

然后就一直找问题在哪儿，最后找到答案是在_config.yml文件中

`url: https://wkq87.github.io/blog

root: /blog/

permalink: :year/:month/:day/:title/

permalink_defaults:`

这个url和root两个字段一定要配置如何配置
